## 財務健康指標

財務健康指標用於評估個人或家庭當前的財務狀況，反映收支管理能力、資產累積效率及財務自由度。

### 1.1 月收支平衡

#### 定義
月收支平衡是評估每月收入與支出之間的關係，反映短期財務管理能力。

#### 計算公式
```
月收支餘額 = 月收入 - 月支出
月儲蓄率 = (月收入 - 月支出) ÷ 月收入 × 100%
```

#### 評估標準
| 狀態 | 條件 | 建議儲蓄率 |
|------|------|-----------|
| 優秀 | 月收入 > 月支出 且 儲蓄率 ≥ 20% | ≥ 20% |
| 良好 | 月收入 > 月支出 且 儲蓄率 10-20% | 10-20% |
| 及格 | 月收入 > 月支出 且 儲蓄率 < 10% | < 10% |
| 不及格 | 月收入 ≤ 月支出 | 負值 |

#### 模組化回應語句

**達成標準（月收入 > 月支出）**
```
狀態：✓ 達成標準

說明：
您的月收入 {monthlyIncome} 元大於月支出 {monthlyExpense} 元，每月有 {monthlyBalance} 元的盈餘（儲蓄率 {savingsRate}%），顯示基本的收支管理良好。

正向影響：
每月盈餘可用於儲蓄、投資或償還債務，有助於改善整體財務狀況。建議將盈餘分配如下：
- 50% 用於建立緊急預備金（目標：3-6個月支出）
- 30% 用於長期投資（退休金、股票、基金等）
- 20% 用於償還高利率債務（信用卡、消費貸款等）

下一步建議：
{根據儲蓄率提供具體建議}
- 若儲蓄率 ≥ 20%：「您的儲蓄率表現優異，請繼續保持，並考慮增加投資比例。」
- 若儲蓄率 10-20%：「您的儲蓄率良好，可考慮檢視非必要支出，進一步提升儲蓄空間。」
- 若儲蓄率 < 10%：「建議檢視支出結構，找出可節省的項目，提升儲蓄率至15%以上。」
```

**未達標準（月收入 ≤ 月支出）**
```
狀態：✗ 未達標準

說明：
您的月支出 {monthlyExpense} 元超過月收入 {monthlyIncome} 元，每月出現 {Math.abs(monthlyBalance)} 元的赤字，需要立即調整支出結構。

負面影響：
月度赤字會消耗存款，增加債務負擔，長期將導致財務惡化，可能陷入「月光族」或「負債循環」的困境。

緊急建議：
1. 立即盤點所有支出項目，區分「必要」與「非必要」支出
2. 削減非必要開支（娛樂、訂閱服務、外食等），目標減少支出 {suggestedCutAmount} 元
3. 尋找增加收入的機會（兼職、加班、技能提升等）
4. 避免使用信用卡循環利息或高利率貸款

行動計畫範例：
- 第1個月：削減 30% 非必要支出
- 第2個月：尋找額外收入來源
- 第3個月：達成收支平衡（月餘額 ≥ 0）
```

### 1.2 年收支平衡

#### 定義
年收支平衡考量年度不規則收入與支出，評估長期財務穩定性。

#### 計算公式
```
年收支餘額 = 年收入 - 年支出
年儲蓄率 = (年收入 - 年支出) ÷ 年收入 × 100%
```

#### 評估標準
| 狀態 | 條件 | 說明 |
|------|------|------|
| 達成標準 | 年收入 > 年支出 | 整體財務穩定 |
| 未達標準 | 年收入 ≤ 年支出 | 需調整年度預算 |

#### 特殊情況說明
- 若「月收支平衡達標」但「年收支未達標」，通常是因為年度大額支出（旅遊、醫療、家電更換等）未列入月度預算
- 建議將年度特殊支出攤提至月度預算中

#### 模組化回應語句

**達成標準（年收入 > 年支出）**
```
狀態：✓ 達成標準

說明：
您的年收入 {yearlyIncome} 元大於年支出 {yearlyExpense} 元，年度盈餘為 {yearlyBalance} 元（年儲蓄率 {yearlySavingsRate}%），整體財務狀況穩定。

正向影響：
年度盈餘有助於資產累積和長期財務目標達成，為退休、購房、子女教育等重大支出提供資金基礎。

建議：
繼續保持良好的年度收支管理，並將盈餘用於長期投資或資產配置優化。
```

**未達標準（年收入 ≤ 年支出）**
```
狀態：✗ 未達標準

說明：
{若月收支平衡達標}
雖然月收支平衡良好，但年支出 {yearlyExpense} 元超過年收入 {yearlyIncome} 元，年度赤字為 {Math.abs(yearlyBalance)} 元，主要原因可能是年度特殊支出（如旅遊、家電更換、醫療費用等）較高。

{若月收支也未達標}
您的年支出 {yearlyExpense} 元超過年收入 {yearlyIncome} 元，年度赤字為 {Math.abs(yearlyBalance)} 元，顯示整體財務結構需要全面調整。

負面影響：
年度赤字需要動用存款或增加借貸，長期會侵蝕資產基礎，影響財務安全。

建議：
1. 設立年度支出預算，將大額支出分散到每月預算中（如每月提撥 {Math.abs(yearlyBalance)/12} 元作為年度支出基金）
2. 尋找增加年收入的機會（年終獎金、投資收益、兼職收入等）
3. 檢視年度支出的必要性，削減非必要的大額開銷
4. 建立「年度支出專戶」，每月固定存入預期的年度大額支出金額
```

### 1.3 資產負債比

#### 定義
資產負債比評估個人或家庭的淨資產狀況，反映財務結構的健康程度。

#### 計算公式
```
淨資產 = 總資產 - 總負債
資產負債比 = 總負債 ÷ 總資產 × 100%
淨資產比 = 淨資產 ÷ 總資產 × 100%
```

#### 資產與負債分類

**總資產包含：**
- 現金與活期存款
- 定期存款
- 股票、基金等投資
- 房地產（市值）
- 其他有價資產

**總負債包含：**
- 房屋貸款
- 汽車貸款
- 信用卡債
- 消費性貸款
- 其他借款

#### 評估標準
| 淨資產狀況 | 資產負債比 | 財務健康度 |
|----------|----------|-----------|
| 淨資產 > 0 | < 50% | 優秀 |
| 淨資產 > 0 | 50-70% | 良好 |
| 淨資產 > 0 | > 70% | 需改善 |
| 淨資產 ≤ 0 | ≥ 100% | 危險 |

#### 模組化回應語句

**達成標準（總資產 > 總負債）**
```
狀態：✓ 達成標準

說明：
您的總資產 {assets} 元大於總負債 {liabilities} 元，淨資產為 {netAssets} 元（淨資產比 {netAssetRatio}%），顯示整體財務結構健康。

資產明細：
{顯示資產組成分析}

負債明細：
{顯示負債組成分析}

正向影響：
正淨資產提供財務安全緩衝，有助於應對突發狀況和未來投資，為財務自由奠定基礎。

建議：
{根據資產負債比提供建議}
- 若資產負債比 < 50%：「您的負債控制良好，建議優化資產配置，增加投資性資產比例。」
- 若資產負債比 50-70%：「建議加速償還高利率債務（信用卡、消費貸款），降低負債比至 50% 以下。」
- 若資產負債比 > 70%：「負債比例偏高，建議優先償還債務，避免新增不必要的負債。」
```

**未達標準（總資產 ≤ 總負債）**
```
狀態：✗ 未達標準（高度警示）

說明：
您的負債 {liabilities} 元超過資產 {assets} 元，淨資產為 {netAssets} 元（資產負債比 {debtRatio}%），財務結構需要緊急調整。

負面影響：
負淨資產表示財務脆弱，任何收入中斷都可能導致財務危機，甚至破產風險。

緊急行動計畫：
1. 【停止舉債】立即停止任何新的借貸行為，避免債務雪球效應
2. 【債務清單】列出所有債務，標註利率、金額、還款期限
3. 【優先順序】優先償還高利率債務（信用卡 > 消費貸款 > 房貸）
4. 【增加收入】尋找額外收入來源，將所有額外收入用於還債
5. 【減少支出】大幅削減非必要支出，進入「緊縮模式」
6. 【專業諮詢】若債務過高，考慮尋求債務協商或財務顧問協助

目標：
- 短期（3個月內）：停止債務增長
- 中期（6-12個月）：淨資產轉為正值
- 長期（2-3年）：資產負債比降至 70% 以下
```

### 1.4 被動收入覆蓋率

#### 定義
被動收入是指不需要投入大量時間和精力即可持續獲得的收入，反映財務自由度。

#### 計算公式
```
被動收入覆蓋率 = 月被動收入 ÷ 月支出 × 100%
被動收入占比 = 月被動收入 ÷ 月總收入 × 100%
```

#### 被動收入來源
- 投資股息、債券利息
- 房地產租金收入
- 版權、專利收入
- 退休金、年金
- 其他持續性非工資收入

#### 評估標準
| 被動收入覆蓋率 | 財務自由度 | 說明 |
|-------------|----------|------|
| ≥ 100% | 完全財務自由 | 被動收入足以覆蓋所有支出 |
| 60-100% | 半財務自由 | 接近財務自由，可選擇性工作 |
| 30-60% | 部分自由 | 有一定保障，但仍需工作收入 |
| < 30% | 初步建立 | 開始建立被動收入 |
| 0% | 未建立 | 完全依賴工作收入 |

#### 模組化回應語句

**已建立被動收入（被動收入 > 0）**
```
狀態：✓ 達成標準

說明：
您有每月 {passiveIncome} 元的被動收入（{被動收入來源說明}），被動收入覆蓋率為 {passiveIncomeCoverageRate}%（占月收入的 {passiveIncomeRatio}%），為未來退休生活提供了{coverage_level}保障。

{coverage_level 判斷}
- 若覆蓋率 ≥ 100%：「完全」
- 若覆蓋率 60-100%：「良好的」
- 若覆蓋率 30-60%：「部分」
- 若覆蓋率 < 30%：「基礎」

正向影響：
被動收入提供財務穩定性，減少對工作收入的依賴，有助於退休規劃和應對失業風險。

進階建議：
{根據覆蓋率提供建議}
- 若覆蓋率 ≥ 100%：「恭喜！您已達成財務自由。建議持續優化投資組合，確保被動收入的穩定性和成長性。」
- 若覆蓋率 60-100%：「您接近財務自由，建議繼續增加被動收入來源，目標達到 100% 覆蓋率。」
- 若覆蓋率 30-60%：「您的被動收入已有基礎，建議持續投資，增加股息收入或租金收入。」
- 若覆蓋率 < 30%：「建議積極增加被動收入來源，考慮定期定額投資高股息股票、債券基金或購買收租房產，目標為月收入的 60% 以上。」

具體行動：
1. 定期定額投資高股息 ETF 或股票（建議每月投資 {suggestedInvestment} 元）
2. 增加債券或債券基金配置，獲取穩定利息收入
3. 考慮購買收租房產（需評估投資報酬率 > 5%）
4. 建立多元化被動收入組合，降低單一收入來源風險
```

**未建立被動收入（被動收入 = 0）**
```
狀態：✗ 未達標準

說明：
您目前沒有被動收入來源，完全依賴工作收入，退休後可能面臨收入中斷的風險。

負面影響：
- 缺乏被動收入增加財務風險，失業或退休後將完全失去收入
- 無法實現財務自由，必須持續工作直到退休
- 退休後可能面臨生活品質大幅下降

緊急建議：
立即開始建立被動收入來源，這是實現財務自由的關鍵步驟。

行動計畫：
1. 【第1階段：學習與規劃（1-3個月）】
   - 學習投資理財知識（股票、基金、債券、房地產）
   - 評估風險承受能力
   - 設定被動收入目標（建議目標：月收入的 60%）

2. 【第2階段：開始投資（3-6個月）】
   - 建立緊急預備金後，開始定期定額投資
   - 每月投資 {建議投資金額} 元於高股息 ETF 或指數基金
   - 若有額外資金，考慮債券或收租房產

3. 【第3階段：持續優化（6個月後）】
   - 定期檢視投資組合表現
   - 逐步增加被動收入比例
   - 目標：5年內達成月收入 30% 的被動收入覆蓋率

預期成果：
假設每月投資 {monthlyInvestment} 元，年化報酬率 6%，預計 {years} 年後可產生月被動收入 {projectedPassiveIncome} 元。
```

## 財務安全指標

財務安全指標用於評估個人或家庭應對風險的能力，包括緊急應變、保險保障、未來規劃及信用健康。

### 2.1 緊急預備金充足度

#### 定義
緊急預備金是用於應對突發狀況（失業、疾病、意外等）的專用資金，確保短期財務安全。

#### 計算公式
```
緊急預備金月數 = 緊急預備金總額 ÷ 月支出
緊急預備金充足度 = (緊急預備金總額 ÷ (月支出 × 目標月數)) × 100%
```

#### 評估標準
| 職業穩定性 | 建議月數 | 目標金額（以月支出5萬為例） |
|----------|---------|--------------------------|
| 高度穩定（公務員、大企業員工） | 3-4個月 | 15-20萬元 |
| 一般穩定（一般上班族） | 4-6個月 | 20-30萬元 |
| 不穩定（自由業、業務性質） | 6-12個月 | 30-60萬元 |

#### 模組化回應語句

**達成標準（緊急預備金 ≥ 3個月支出）**
```
狀態：✓ 達成標準

說明：
您的緊急預備金 {emergencyFund} 元可支撐 {emergencyMonths} 個月的支出（月支出 {monthlyExpense} 元），{達成評價}。

{達成評價判斷}
- 若月數 ≥ 6：「已達到優秀水準」
- 若月數 4-6：「符合一般標準」
- 若月數 3-4：「剛好達到最低標準」

正向影響：
充足的緊急預備金提供財務安全網，讓您能從容應對突發狀況（失業、醫療、意外等），避免被迫借貸或變賣資產。

維護建議：
1. 將緊急預備金存放於「高流動性」帳戶（活存、貨幣基金、短期定存）
2. 定期檢視金額是否需要調整（若支出增加，應同步提高預備金）
3. 僅在真正緊急情況下動用，事後立即補足
4. {若月數 < 6}「建議繼續提升至 6 個月支出（{targetAmount} 元），提供更完善的保障。」
```

**未達標準（緊急預備金 < 3個月支出）**
```
狀態：✗ 未達標準（重要警示）

說明：
您的緊急預備金 {emergencyFund} 元僅能支撐 {emergencyMonths} 個月的支出（月支出 {monthlyExpense} 元），低於建議的最低標準（3-6個月）。

{嚴重程度判斷}
- 若月數 < 1：「嚴重不足，財務極度脆弱」
- 若月數 1-2：「明顯不足，急需改善」
- 若月數 2-3：「略為不足，建議盡快補足」

負面影響：
緊急預備金不足可能導致：
- 突發狀況時需要緊急借貸（高利率信用卡、貸款）
- 被迫變賣資產（房產、股票等），可能虧損出場
- 財務壓力影響工作和生活品質
- 陷入「借新還舊」的債務循環

緊急行動計畫：
目標：在 {建議月數} 個月內建立 {targetAmount} 元的緊急預備金（3個月支出）

【優先級：最高】
1. 第1個月：立即存入 {firstMonthTarget} 元作為啟動金
2. 每月固定提撥 {monthlyContribution} 元（建議為月收入的 10-15%）

### 2.2 保險保障完整性

#### 定義
保險保障用於轉移重大財務風險（疾病、意外、死亡等），保護個人及家庭經濟穩定。

#### 基本保險配置
```
基礎保障（必備）：
- 醫療險：住院、手術、重大疾病
- 意外險：意外身故、失能、醫療

完整保障（建議）：
+ 壽險：家庭經濟支柱必備（保額 = 年收入 × 5-10倍）
+ 失能險：因疾病或意外導致失能的收入保障
+ 癌症險：針對癌症治療的特別保障
```

#### 保額評估公式
```
壽險建議保額 = (家庭年支出 × 10) + 未償還負債 + 子女教育金
意外險建議保額 = 年收入 × 5-10倍
醫療險建議保額 = 每日病房費 1,500-3,000元 + 手術限額 10-30萬
```

#### 評估標準
| 保險配置 | 評估 | 說明 |
|---------|-----|------|
| 有醫療險 + 意外險 + 壽險 + 失能險 | 優秀 | 保障完整 |
| 有醫療險 + 意外險 + 壽險 | 良好 | 基本完整，建議增加失能險 |
| 有醫療險 + 意外險 | 及格 | 達到基本保障 |
| 僅有單一險種 | 不足 | 保障不完整 |
| 無任何保險 | 危險 | 完全暴露於風險中 |

### 2.3 未來財務規劃

#### 定義
未來財務規劃是針對重大人生目標（退休、購房、子女教育等）預先制定的財務準備計畫。

#### 常見財務規劃項目
```

- 進修學習

中期目標（3-10年）：

- 子女教育金
- 創業基金

長期目標（10年以上）：
- 退休金準備
- 遺產規劃
- 長期照護準備
```

#### 財務規劃公式
```
目標金額計算（考慮通膨）：
未來所需金額 = 現值 × (1 + 通膨率)^年數

每月應儲蓄金額計算：
月儲蓄額 = 目標金額 × 月利率 ÷ ((1 + 月利率)^月數 - 1)

其中：
- 通膨率建議值：2-3%
- 投資報酬率建議值：3-6%（保守）、6-10%（積極）
```

#### 評估標準
| 規劃狀況 | 評估 |
|---------|-----|
| 有明確目標、金額、時程及執行計畫 | 優秀 |
| 有目標和概略金額但無具體計畫 | 良好 |
| 有模糊想法但無具體目標 | 不足 |
| 完全沒有未來規劃 | 未達標 |

### 2.4 信用評分健康度

#### 定義
信用評分（信用分數）是金融機構評估個人信用風險的重要指標，影響貸款核准率和利率。

#### 信用評分計算因素
```
影響因素（權重）：
1. 還款紀錄（35%）：是否按時還款、有無逾期紀錄
2. 負債比例（30%）：信用卡使用率、貸款餘額
3. 信用歷史長度（15%）：信用往來時間
4. 新信用申請（10%）：近期申請信用卡、貸款次數
5. 信用組合（10%）：信用卡、貸款等多樣性
```

#### 評估標準（台灣聯徵中心評分系統）
| 信用分數範圍 | 評等 | 貸款影響 |

| 600-699 | 普通 | 核准率中等，利率正常 |
| 500-599 | 不佳 | 核准率低，利率較高 |
| < 500 | 極差 | 核准率極低，可能被拒絕 |

#### 改善信用評分公式
```
最佳信用卡使用率 = 已使用額度 ÷ 總額度 < 30%
負債收入比 = 每月負債支出 ÷ 月收入 < 40%

## 評分計算方式

### 財務健康總分計算
```
財務健康分數 = Σ(各指標分數)
最高分：100分（每項指標25分）

計算規則：
1. 月收支平衡：月收入 > 月支出 → +25分
2. 年收支平衡：年收入 > 年支出 → +25分
3. 資產負債比：總資產 > 總負債 → +25分
4. 被動收入：有被動收入（>0） → +25分
```

### 財務安全總分計算
```
財務安全分數 = Σ(各指標分數)
最高分：100分（每項指標25分）

計算規則：
1. 緊急預備金：緊急預備金 ≥ 3個月支出 → +25分
2. 保險保障：有醫療險 + 意外險 → +25分
3. 未來規劃：有明確財務規劃 → +25分
4. 信用評分：信用分數 ≥ 700分 → +25分
```

### 綜合評級
| 總分範圍 | 評級 | 財務狀況 |
|---------|-----|---------|
| 150-200分 | A+ | 財務非常健康 |
| 100-149分 | A | 財務健康 |
| 75-99分 | B | 財務尚可，需改善 |
| 50-74分 | C | 財務不佳，需積極改善 |
| < 50分 | D | 財務危機，需緊急處理 |

## 模組化回應語句設計

### 設計原則
1. **條件判斷式結構**：根據數值範圍自動選擇對應語句
2. **變數替換**：使用 `{變數名}` 格式動態帶入數據
3. **分級回應**：根據嚴重程度提供不同深度的建議
4. **行動導向**：提供具體可執行的步驟，而非空泛建議

### 模組化範例：月收支平衡

```javascript
// 模組化回應函數
function getMonthlyBalanceResponse(monthlyIncome, monthlyExpense) {
  const balance = monthlyIncome - monthlyExpense;
  const savingsRate = (balance / monthlyIncome * 100).toFixed(1);
  
  // 條件判斷
  if (balance > 0) {
    // 達成標準
    let savingsLevel;
    let suggestion;
    
    if (savingsRate >= 20) {
      savingsLevel = "優秀";
      suggestion = "您的儲蓄率表現優異，請繼續保持，並考慮增加投資比例。";
    } else if (savingsRate >= 10) {
      savingsLevel = "良好";
      suggestion = "您的儲蓄率良好，可考慮檢視非必要支出，進一步提升儲蓄空間。";
    } else {
      savingsLevel = "及格";
      suggestion = "建議檢視支出結構，找出可節省的項目，提升儲蓄率至15%以上。";
    }
    
    return {
      status: "✓ 達成標準",
      description: `您的月收入 ${monthlyIncome.toLocaleString()} 元大於月支出 ${monthlyExpense.toLocaleString()} 元，每月有 ${balance.toLocaleString()} 元的盈餘（儲蓄率 ${savingsRate}%），顯示基本的收支管理良好。`,
      impact: "每月盈餘可用於儲蓄、投資或償還債務，有助於改善整體財務狀況。",
      suggestion: `儲蓄率評級：${savingsLevel}\n\n${suggestion}\n\n建議分配：\n- 50% 用於緊急預備金\n- 30% 用於長期投資\n- 20% 用於償還高利率債務`
    };
  } else {
    // 未達標準
    const deficit = Math.abs(balance);
    const suggestedCut = Math.ceil(deficit * 1.2); // 建議削減金額（含緩衝）
    
    return {
      status: "✗ 未達標準",
      description: `您的月支出 ${monthlyExpense.toLocaleString()} 元超過月收入 ${monthlyIncome.toLocaleString()} 元，每月出現 ${deficit.toLocaleString()} 元的赤字，需要立即調整支出結構。`,
      impact: "月度赤字會消耗存款，增加債務負擔，長期將導致財務惡化。",
      suggestion: `緊急行動計畫：\n1. 立即削減非必要支出 ${suggestedCut.toLocaleString()} 元\n2. 尋找增加收入的機會\n3. 目標：3個月內達成收支平衡\n4. 避免使用信用卡循環利息`
    };
  }
}
```

### 模組化範例：緊急預備金

```javascript
function getEmergencyFundResponse(emergencyFund, monthlyExpense, jobStability) {
  const months = emergencyFund / monthlyExpense;
  
  // 根據職業穩定性設定目標月數
  let targetMonths;
  switch(jobStability) {
    case 'high':
      targetMonths = 3;
      break;
    case 'medium':
      targetMonths = 6;
      break;
    case 'low':
      targetMonths = 12;
      break;
    default:
      targetMonths = 6;
  }
  
  const targetAmount = monthlyExpense * targetMonths;
  
  if (months >= 3) {
    // 達成標準
    let achievement;
    if (months >= 6) {
      achievement = "已達到優秀水準";
    } else if (months >= 4) {
      achievement = "符合一般標準";
    } else {
      achievement = "剛好達到最低標準";
    }
    
    return {
      status: "✓ 達成標準",
      description: `您的緊急預備金 ${emergencyFund.toLocaleString()} 元可支撐 ${months.toFixed(1)} 個月的支出，${achievement}。`,
      suggestion: months < 6 
        ? `建議繼續提升至 6 個月支出（${(monthlyExpense * 6).toLocaleString()} 元）。`
        : "維持目前水準，定期檢視是否需調整。"
    };
  } else {
    // 未達標準
    const gap = targetAmount - emergencyFund;
    const monthlyContribution = Math.ceil(gap / 6); // 假設6個月完成
    
    let severity;
    if (months < 1) {
      severity = "嚴重不足，財務極度脆弱";
    } else if (months < 2) {
      severity = "明顯不足，急需改善";
    } else {
      severity = "略為不足，建議盡快補足";
    }
    
    return {
      status: "✗ 未達標準",
      description: `您的緊急預備金 ${emergencyFund.toLocaleString()} 元僅能支撐 ${months.toFixed(1)} 個月的支出。${severity}。`,
      actionPlan: `
目標：建立 ${targetAmount.toLocaleString()} 元緊急預備金（${targetMonths}個月支出）
差距：${gap.toLocaleString()} 元

行動計畫：
1. 立即存入初期啟動金
2. 每月固定提撥 ${monthlyContribution.toLocaleString()} 元
3. 所有意外收入優先補足此目標
4. 預計 6 個月完成

進度追蹤：
- 3個月後：${(monthlyContribution * 3).toLocaleString()} 元（${(monthlyContribution * 3 / targetAmount * 100).toFixed(0)}%）
- 6個月後：${(monthlyContribution * 6).toLocaleString()} 元（100%）`
    };
  }
}
```
